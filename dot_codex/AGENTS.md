# Global Agent Guidance

## Plan Mode

- Make the plan extremely concise. Sacrifice grammar for the sake of concision.
- At the end of each plan, give me a list of unresolved questions to answer, if any.

## PowerShell Command Formatting

- Avoid Markdown inline code for runnable PowerShell commands because copied backticks may be treated as escape characters.
- Prefer plain command lines in prose or fenced `text` blocks when sharing commands for copy/paste.
- If a command is being discussed rather than presented for execution, normal prose is sufficient.
